The isNaN() function, evaluates whether or no a variable is not a number. If you pass a non-number variable, such as a string into it, it will return true.

Boolean logic is facilitated in programming, through the passage of true false instructions through logical gates. Some examples below...

(7 + 12) * 14 - 1243 = *see console for answer*

JavaScript gives us the == operator, to allow us to compare variables, and return a boolean of whether or not they are equivalent.

Additionally, JavaScript gives us the === operator, which returns a boolean value, relative to whether or not the variables are equal in both type and value.

Some other common logic operators are AND (&&) and OR (||)

The next operator is NOT, symbolized by '!'. NOT can be used to make any boolean value, have its opposite value.
